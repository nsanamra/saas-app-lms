# Document

## Setup 
1. `npx create-next-app@latest ./`

2. For shadcn : `npx shadcn@latest init`  

## Adding buttons in shadcn 
- `npx shadcn@latest add button` 

### For making a new route :

3. Then you go on http://localhost:3000/sign-in
4. If we have a route in the url then name the folder like this `[params...]` and then make a file page.tsx inside it.
